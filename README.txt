Requerimientos de bibliotecas utilizadas:

- Versión Python: 3.11.11
- Entorno de trabajo: Spyder IDE v6.0.4
- Requisitos mínimos:
	pandas
	scikit-learn
	matplotlib
	seaborn
	numpy
	duckdb
Se puede instalar copiando en la consola del entorno (sin las comillas):
"pip install pandas scikit-learn matplotlib seaborn numpy duckdb itertools"

Instrucciones para manejar el archivo "TP-Grupo_09.py":

1. Situar el entorno de trabajo en la carpeta en donde se encuentre el archivo .py.

2. Ejecutar el bloque IMPORTACIÓN DE LIBRERIAS.

3. Ejecutar el bloque CARGA DE DATOS.

4. Ejecutar el bloque FUNCIONES DEFINIDAS.

5. Ejecutar los bloques correspondientes a cada ítem de la sección "Ejercicios" (enunciado del TP).
Estos bloques son INDEPENDIENTES entre sí(no poseen un orden específico ni tampoco se 
requiere la ejecución previa de uno para el correcto funcionameniento de otro).
